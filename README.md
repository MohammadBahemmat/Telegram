# 🔄 V2ray Apps Update Checker

<div dir="rtl">

**ربات بررسی و اعلام نسخه‌های جدید نرم‌افزارهای V2ray در کانال تلگرام**  

این پروژه با استفاده از **GitHub Actions** به‌طور خودکار هر ساعت (یا با زمان‌بندی دلخواه) آخرین نسخه‌های منتشر شده از نرم‌افزارهای زیر را بررسی می‌کند و در صورت وجود آپدیت جدید، پیامی حاوی لینک دانلود و راهنمای نصب به **کانال تلگرام** شما ارسال می‌کند:

- 🖥️ **v2rayN** (نسخه دسکتاپ ویندوز و مک)  
- 📱 **v2rayNG** (نسخه اندروید)  
- 🍏 **یادآوری اپ‌های iOS** (FoXray, Streisand, V2Box)  

هدف از ساخت این ربات، اطلاع‌رسانی سریع و دقیق به کاربران فارسی‌زبان است تا هیچ آپدیت مهمی را از دست ندهند و همیشه آخرین نسخه‌ها را در دسترس داشته باشند.

---

## 🚀 ویژگی‌ها

- **بررسی خودکار** از طریق GitHub Actions (بدون نیاز به سرور اختصاصی)  
- **تشخیص لینک‌های مناسب** برای هر پلتفرم (Self-Contained ویندوز، arm64 اندروید و غیره)  
- **ارسال پیام به تلگرام** با فرمت HTML و دیس‌ایبل کردن پیش‌نمایش لینک‌ها  
- **ذخیره وضعیت آخرین نسخه** برای جلوگیری از ارسال تکراری  
- **راهنمای آپدیت** داخل پیام برای کاربران  
- **پشتیبانی از iOS** با لینک مستقیم به اپ استور  
- **کاملاً رایگان و اپن‌سورس**

## ⚙️ نحوه استفاده (برای توسعه‌دهندگان)

اگر می‌خواهید این پروژه را روی مخزن خودتان اجرا کنید، مراحل زیر را دنبال کنید:

### ۱. Fork کردن پروژه
ابتدا این مخزن را Fork کنید.

### ۲. تنظیم Secrets
به بخش `Settings > Secrets and variables > Actions` بروید و دو Secret زیر را ایجاد کنید:

| نام | توضیح |
|----------|--------|
| `BOT_TOKEN` | توکن ربات تلگرام (از [BotFather](https://t.me/BotFather) دریافت کنید) |
| `CHANNEL_ID` | شناسه عددی کانال تلگرام (مثلاً `123456789@-`) |

### ۳. فعال‌سازی GitHub Actions
مطمئن شوید که GitHub Actions در مخزن شما فعال است. workflow به صورت پیش‌فرض هر یک ساعت یکبار اجرا می‌شود که می‌توانید زمان‌بندی آن را در فایل `.github/workflows/main.yml` تغییر دهید.

### ۴. بررسی فایل وضعیت
فایل `last_versions.json` به صورت خودکار ایجاد می‌شود و نیازی به تغییر دستی آن نیست.

## 🛠️ سفارشی‌سازی

- برای **تغییر فرکانس بررسی**، مقدار cron را در `main.yml` ویرایش کنید (مثلاً `'0 */6 * * *'` برای هر ۶ ساعت).  
- می‌توانید **نرم‌افزارهای مورد نظر** را در دیکشنری `REPOS_TO_CHECK` و `IOS_APPS` داخل `version_checker.py` اضافه یا حذف کنید.  
- **متن پیام** یا **کلمات کلیدی جستجوی فایل** نیز در همین فایل قابل تغییر است.
